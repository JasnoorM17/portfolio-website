About the Site

Mindset focuses on clear structure and readable typography. I used WordPress templates, parts, and simple customizations to keep the site fast and easy to maintain.

Reflections

How did I feel?

Overall I’m happy with this site since it worked the way I expected and matched the look I wanted.

What did I learn?

  • How to add and modify custom parts in WordPress to control layout and content sections.
  • How to use plugins like WPForms to collect submissions with a simple setup and consistent styling.
  • How to navigate WordPress and get the site working in both local and live environments.
  • Using WP Migrate to sync content and settings between local and live.

Key Takeaways

  • Start with core templates, then add custom parts where more control is needed.
  • Plan media sizes and alt text early for performance and accessibility.
