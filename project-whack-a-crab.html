About the Game

Whack A Crab is a lighthearted browser game inspired by the classic “Whack-a-Mole.” Players click fast-moving crabs as they pop up to score points before the timer ends. It’s built with vanilla JavaScript for logic (timers, score tracking, event handling) and CSS for animations and responsiveness.

Reflections

How did I feel?

I’m proud of this being my first completed game project. It was fun to see my JavaScript knowledge come to life in a working, interactive way.

What did I learn?

  • Core JavaScript skills: timers, event listeners, and DOM manipulation for score/time updates.
  • How responsive feedback makes games feel satisfying — quick clicks, fast animations, smooth flow.
  • That small games benefit from organized functions and clear state management more than complex frameworks.

Key Takeaways

  • Improved confidence working with vanilla JS and understanding browser event cycles.
  • Learned the importance of accessibility and feedback in interactive design.
  • Would improve hit detection and add progressive difficulty, sound effects, and local high scores.
  • Ensured the game is mobile-friendly and performs smoothly across devices.
